Welcome to Hyrule Warriors Helper!

This project is an experiment and for fun. It aims to provide help with the adventure mode of Hyrule Warriors Definitive Edition.


Features

Currently this website provides interactive maps for the following Adventure Mode stages:

  • Adventure Map
  • Great Sea Map
  • Twilight Map
  • Termina Map

These interactive maps provide the following features:

  • Details of every tile, like rewards, requirements, treasure and more
  • Search details of every tile on location and items
  • Map-wide fulltext search to find character-, weapon-, or item-specific tiles

Planned Features:

  • Interactive Maps for the missing stages
  • Links to character guides
  • Detailed skulltula token descriptions for each tile
  • Display good farming locations for materials
